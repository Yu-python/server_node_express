# SERVER_NODE_EXPRESS

Node + Express + Mariadb 提供后台接口服务。

### Run Project

``` bash
$ git clone git@github.com:py-novel/server_node_express.git
$ cd server_node_express
$ yarn
$ npm run ts:watch
$ npm run dev
```

访问 `localhost:4000/api/v3`。
